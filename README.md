Student Information System (SIS)
About
The Student Information System (SIS) is a C++ based backend application designed to manage student data, academic progress, and course records. This project focuses on implementing robust data structures and logic to handle CRUD (Create, Read, Update, Delete) operations efficiently, ensuring data integrity through strict input validation.

Key Features
Student Management: Add, search, update, and delete student profiles with validation for IDs, phone numbers, and dates of birth.

Data Integrity: Implements setter validation to ensure all student and course data meets university standards.

Persistent Storage: Automatically saves and loads data to/from local text files (students.txt), ensuring records are preserved between sessions.

Performance-Focused: Designed with a backend-first approach to manage complex record keeping without relying on GUI overhead.

Technologies Used
Language: C++

IDE: Visual Studio

Concepts: Object-Oriented Programming (OOP), File I/O, Data Structures (Vectors), Algorithmic Validation.

How to Build and Run
Clone the repository: Download the project files to your local machine.

Open in Visual Studio: Launch Visual Studio and open the project solution file (.sln).

Build: Navigate to the Build menu and select Build Solution.

Run: Press F5 or the Start button to execute the application.

Data Files: Ensure the application has read/write permissions in the directory where the .txt files are located to allow data persistence.
